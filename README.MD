# Herramientas necesarias
- node LTS 24.16.0

# Procedimiento para hacer el build de producción

### Realizar la instalación de paquetes
```bash
$ npm install
```

### Hacer el build de producción y levantamiento del servidor express
```bash
$ npm run start
```
### **El servidor se abre en el puerto 3500**


### Abrir el dashboard
El dashboard se encuentra en: http://localhost:3500/static/dashboard.html
